传送到手表

已下载文件：courses.json  → 保存在手机 Download 文件夹
选一种方式传到手表
推荐：手机 HTTP Server
1. Play商店搜 "Simple HTTP Server" 安装
2. 打开 App → 文件夹图标 → 选 Download → 启动
3. App 显示地址如：
http://192.168.x.x:8080
4. 手表 Settings → 网络下载课表 → 输入:
http://192.168.x.x:8080/courses.json
备选：GitHub Gist
1. 打开 gist.github.com → 粘贴 JSON
2. 文件名 courses.json → Create → 点 Raw
3. 复制 Raw 页面 URL → 手表下载
应急：粘贴导入 — 手表 Settings → 粘贴导入 → 粘贴下方 JSON
